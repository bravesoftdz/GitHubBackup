This software and GitHub API wrapper were written by Jerry Dodge. It in itself is open-source on GitHub, available for anybody to contribute.

Embarcadero Delphi 10.1 Berlin is the base language for this software.

Installation is compliments of Inno Setup.

Help authoring is with the help of WinCHM.

OpenSSL binaries are used for HTTPS.

There are numerous third-party Delphi libraries involved...

  • Vcl Styles Utils
  • Virtual TreeView
  • Indy Internet Components
  • Super Object
  • Font Awesome
