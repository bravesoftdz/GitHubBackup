The basis of the main screen is to show you a list of all repositories you have access to in a given user or organization account. Click on the "Refresh" button to perform the search. This screen will list you all repositories which the account token you entered can access from the selected user or organization.

  • Refresh - Performs search for GitHub repositories
  • Filter - Narrows the list down to partial repository name matches
  • Sort - Choose which column you wish to sort by, as well as Ascending or Descending
