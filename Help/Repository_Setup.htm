You will need to choose which GitHub account you wish to work with, as well as whether it's a user or organization account. Do this by creating different profiles for different accounts.

  • Add - Press this button to add a new profile.
  • Delete - Press this button to delete the selected profile. Note you must have at least one.
  • Title - A user-friendly title for the profile.
  • Account Name - The unique GitHub account name which you want to work with
  • Account  Type - The type of account you entered, whether user or organization.
  • Download Directory - The folder on your computer or local network where you want the files to be downloaded to.

Note: The first time using this app, it will automatically generate a new sample profile. You may choose to overwrite this, or create more.

Note: If you are upgrading from a version before profiles, you will have a "Default" one automatically converted for you instead.
