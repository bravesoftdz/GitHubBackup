You will need to choose which GitHub account you wish to work with, as well as whether it's a user or organization account. You will also need to choose a directory where you want the files to be downloaded to.

  • Backup Directory - The folder on your computer or local network where you want the files to be downloaded to.
  • Account to List - The unique GitHub account name which you want to work with
  • User / Organization - The type of account you entered.
  • Page Size - Number of repositories to fetch at a time (Not Supported)
