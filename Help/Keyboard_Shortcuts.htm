There are various different keyboard shortcuts which allow you to quickly interact with the application without using the mouse. These are detailed below.

  • F1 - Opens the help file to whatever topic is currently visible/focused.
  • Ctrl + S - Enters the application setup module.
  • F5 - Refreshes the repository list.
  • Ctrl + A - Checks all repositories in the list.
  • Ctrl + N - Unchecks all repositories in the list.
  • Ctrl + L - Checks or unchecks currently highlighted repository.
  • Ctrl + D - Initiates the actual download.
  • Esc - Cancels the current download which is in progress.
  • Ctrl + B - Opens the application's about dialog.
